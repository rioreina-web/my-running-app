## Quick map

- `colors_and_type.css` — paste into any HTML artifact; gives you the full token system
- `fonts/` — Crimson Pro variable + PT Serif (Regular/Italic/Bold)
- `assets/` — Logos and brand marks
- `ui_kits/ios_app/` — Pixel-faithful iOS app recreation; lift the components for any mock that needs iOS-shell screens
- `preview/` — Design-system specimen cards

## The four things to get right

1. **Voice first.** Editorial diary + coach. Spare. Second-person. Period after standalone headlines. Middle dot (`·`) as separator. No emoji, no exclamation points, no "AI-powered."
2. **One coral, used like punctuation.** Never a fill — only the eyebrow color, the one active-state, the inline link.
3. **Three fonts, sharply assigned.** Crimson Pro display, PT Serif body, monospaced for *every* uppercase label.
4. **Editorial rule** (`line · dot · line`) for section breaks. Hairlines for everything else. No card-in-card. No gradients.
