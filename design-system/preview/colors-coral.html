The rule

One coral element per visual cluster, max. If two would compete, drop one to ink-2. Coral is a pointer, never a paint — never a large fill, never a background.

"What about success / error / warning?"

→State doesn't share coral's job. The system has a separate, muted mood palette (green · sage · amber · terracotta · rose · plum) for moods, zones, and load deltas — always at low chroma, always as a tracked pill.
